== express 安装和运行
.安装 express 4 的最新版本
[source,console]
----
pnpm add express@4
----


.查看已安装的 express 版本
[source,console]
----
pnpm list express
----


.运行 express 写的程序
[source,console]
----
# 直接运行
node app.js

# 或使用 nodemon（自动重启，开发时推荐）
pnpm add -D nodemon

# 当前项目的 node_modules/.bin/（本地依赖）
# 全局安装的模块（如果没找到本地版本）
# 如果都没找到，会临时下载并执行（然后自动删除）
npx nodemon app.js
----

NOTE: app.js 是你的应用程序

== express 入门代码
[source,javascript]
----
import express from 'express';
const app = express();

app.get('/', (req, res) => {
    res.send("Hello World!");  // Express 自动处理 Content-Type 和状态码
});

app.listen(8081, () => {
    console.log('服务器正在 8081 端口上监听');
});
----

运行 express 程序

[source,console]
----
npx nodemon helloworld_express.js
----

打开浏览器测试: http://127.0.0.1:8081
